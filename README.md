# Retail-Sales-Performance-Dashboard

📊 Retail Sales Dashboard | Power BI


1. Project Title

Retail Sales Dashboard – Power BI Business Intelligence Project


2. Short Description / Purpose

This project presents an interactive Retail Sales Dashboard built in Power BI to analyze sales performance, profitability, customer behavior, and product trends. The dashboard helps business users monitor key performance indicators (KPIs), identify high-performing products and regions, and make data-driven decisions through dynamic visualizations.


 3. Tech Stack

Power BI Desktop – Dashboard development
Power Query – Data cleaning and transformation
DAX (Data Analysis Expressions) – Calculated columns and measures*Microsoft Excel** – Dataset source
Data Modeling – Relationships between tables
Interactive Visualizations – Charts, KPIs, slicers, and filters


 4. Data Source

Dataset: Retail Sales Dataset (Excel)
Contains information such as:

  * Order ID
  * Order Date
  * Customer
  * Product Category
  * Product Name
  * Region
  * Sales
  * Quantity
  * Profit
  * Discount



5. Features / Highlights

✅ Dashboard Features

* KPI Cards for:

  * Total Sales
  * Total Profit
  * Total Orders
  * Profit Margin
* Monthly Sales Trend Analysis
* Sales by Category
* Sales by Region
* Top Performing Products
* Customer-wise Sales Analysis
* Interactive Filters (Year, Category, Region)
* Dynamic charts with drill-down capability


✅ Business Questions Answered

* What is the total revenue generated?
* Which product category contributes the most sales?
* Which region performs the best?
* What are the monthly sales trends?
* Which products generate the highest profit?
* Which customers contribute the highest sales?
* How does profit margin vary across categories?


 ✅ Business Impact & Insights

* Identified the most profitable products and regions.
* Helped understand seasonal sales patterns.
* Highlighted low-performing categories for improvement.
* Enabled quick decision-making through interactive dashboards.
* Improved reporting efficiency by replacing manual reports with automated visualizations.
